Amigo Secreto — Grupo Piaia
Bem-vindo ao sistema de Amigo Secreto desenvolvido para o Grupo Piaia! Este projeto foi criado para conectar colaboradores com alegria e confiança, proporcionando uma experiência divertida, organizada e segura para o sorteio de amigos secretos.

Teste: https://marceloferrarigit.github.io/amigo-secreto/

Funcionalidades
   Sorteio automático com prevenção de autoatribuição
   Controle de acesso por fila (um participante por vez)
   Geração automática de PDF com resultado personalizado
   Interface amigável e temática natalina
   Lista de participantes atualizada em tempo real
   Reset protegido por senha
   
Como usar
    Acesse o site publicado via GitHub Pages ou Netlify
    Informe seu nome e senha
    Aguarde sua vez na fila
    Escolha um número disponível


Veja o resultado e baixe seu PDF com o amigo secreto
  Regras
  Apenas colaboradores cadastrados podem participar
  Cada número só pode ser usado uma vez
  Cada colaborador só participa uma vez
  O sorteio garante que ninguém tire a si mesmo
  O amigo secreto de cada pessoa é único e não se repete

Tecnologias utilizadas
  HTML, CSS e JavaScript puro

localStorage para persistência de dados
  jsPDF para geração de PDF
